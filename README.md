# 박성환 GitHub Pages 포트폴리오

Java/Spring 백엔드 프로젝트 경험을 정리한 정적 포트폴리오 사이트입니다.

## 권장 공개 URL

GitHub 사용자 페이지 저장소로 운영할 경우:

```text
https://seonghwan01.github.io
```

프로젝트 페이지로 운영할 경우:

```text
https://seonghwan01.github.io/seonghwan01.github.io/
```

## 로컬 미리보기

```bash
cd seonghwan01.github.io
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 확인합니다.

## 구성

- `index.html`: 포트폴리오 본문
- `styles.css`: 반응형 스타일
- `script.js`: 상단 이동 버튼과 앵커 스크롤
- `assets/img/`: 프로젝트 및 프로필 이미지
