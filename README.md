# 박성환 GitHub Pages 포트폴리오

Java/Spring 백엔드 프로젝트 경험을 정리한 정적 포트폴리오 사이트입니다.

## 권장 공개 URL

GitHub 사용자 페이지 저장소로 운영할 경우:

```text
https://seonghwan01.github.io
```

프로젝트 페이지로 운영할 경우:

```text
https://seonghwan01.github.io/seonghwan01.github.io/
```

## 로컬 미리보기

```bash
cd seonghwan01.github.io
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`으로 확인합니다.

## 구성

- `index.html`: 포트폴리오 본문
- `styles.css`: 반응형 스타일
- `script.js`: 상단 이동 버튼과 앵커 스크롤
- `assets/img/`: 프로젝트 및 프로필 이미지

## 공개 저장소 주의

현재 상위 작업 폴더에는 개인 문서, 발표 원본, ZIP 파일이 함께 있습니다.
GitHub 공개 저장소에는 이 폴더 전체가 아니라 `seonghwan01.github.io` 폴더의 파일만 올려야 합니다.

## 확인 항목

- 시크릿 창에서 링크가 열리는지 확인
- 모바일 폭에서 가로 스크롤이 없는지 확인
- GitHub 링크가 404가 아닌지 확인
- 프로젝트 성과가 검증 가능한 근거 중심인지 확인
- 팀 성과와 본인 역할이 섞여 보이지 않는지 확인
